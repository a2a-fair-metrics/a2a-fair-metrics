58-rda-r1-01m-t7-type-and-no-about

This landing page has these FAIR Signposting HTTP headers:

  • rel=cite-as to w3id PID
  • rel=type to https://schema.org/Dataset
  • rel=item to CSV file (text/csv, type attribute declared)

This is a test for FAIR principle RDA-R1-01M - Plurality of accurate and relevant attributes are provided to allow reuse.

Test 7: If this is a landing page (text/html) with separate rel=item, then rel=type includes https://schema.org/AboutPage or http://schema.org/AboutPage
