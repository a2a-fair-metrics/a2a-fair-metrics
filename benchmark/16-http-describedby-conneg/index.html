16-http-describedby-conneg

This landing page has two corresponding HTTP headers:

  • describedby content-negotiated to metadata.ttl (text/turtle, type attribute declared)
  • describedby content-negotiated to metadata.jsonld (application/ld+json, type attribute declared)

The client is forced to use HTTP content negotiation in this scenario, the server will add a Content-Location of the seleted resource.
