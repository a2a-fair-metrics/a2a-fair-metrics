09-http-describedby-citeas-linkset-json-txt

This landing page has four HTTP headers:

  • describedby to index.ttl (text/turtle, type attribute not declared)
  • cite-as to w3id
  • linkset to linkset.json of type application/linkset+json that links to test-apple-data.csv and the above
  • linkset to linkset.txt as application/linkset alternative to the above
There is a Link rel=alternate between linkset.json and linkset.txt
